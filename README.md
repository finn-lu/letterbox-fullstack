# letterbox-fullstack
Reimagination of the Movie Rating and Discovering Platform "Letterboxd"

Project: Letterboxed Full Stack App
Extra Features:
Tinder Swipe Mechanik für Filmauswahl
Movie Wrapped
Integration von Serien und Filmen + Anzeige wo diese Gestreamed werden
Rating und Komment funktion
Suchfuktion
Listenerstellung
Movie Recommendation unterliegend




TECH STACK:
Frontend: Next.js mit TypeScript + Tailwind CSS
Backend: FastAPI
Datenbank: PostgreSQL über Supabase
Externe API: Movie Database (TMDB API)

Deployment:
Vercel
Railway/Render
