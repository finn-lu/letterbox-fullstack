# Letterbox Frontend

Next.js frontend for the Letterboxd reimagination project.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Development
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts
- `npm run dev` start dev server
- `npm run build` production build
- `npm run start` run production server
- `npm run lint` run ESLint
